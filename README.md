# Dynamic UI Blocks (Laravel Project)

## Features
- Admin can create UI blocks
- Enable / Disable blocks
- Reorder blocks
- Dynamic rendering on client side

## Tech Stack
- Laravel
- Blade
- MySQL

## Setup
1. git clone ...
2. composer install
3. cp .env.example .env
4. php artisan migrate --seed //admin Seeder and User Seeder Presant
5. php artisan serve
